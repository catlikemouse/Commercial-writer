# 第36章：周字第三笔？——谁签收，谁就是刽子手

问罪台上，倒计时像一把钉在喉头的刀。

【29:17】

黑墙补出来的那行字，比刀还凉。

【补充：处置执行需交付等价证据】
【等价证据：一笔“周”字第三笔】

周无昼头顶欠冠“嗡”地一震，像被人用指甲在骨头上刮了一下。

他盯着那三个字——第三笔——喉结滚动得发疼。

“你知不知道那是什么？”

裴谦很诚恳：“不知道。”

周无昼差点吐血：“你不知道你还一脸捡到宝？”

裴谦更诚恳：“我只知道它能让我合法去死。”

执案官已经崩到声音发抖：“证人！处置窗口成立，不代表你可以把‘处置’当爱好！”

裴谦疑惑：“你们天天处置别人，今天轮到我，你急什么？”

执案官：“……”

周无昼压着嗓子，像怕那滴墨听见：“第三笔不是墨水，是命格押着的那一笔。周字两笔已经写在我头顶，第三笔一落……我可能就不是周无昼了。”

裴谦眼睛瞬间更亮：“那太好了，你也能重开。”

周无昼一口气差点没上来：“我不想重开！”

灰雾里，司晦承书的声音冷冷插进来：“他不是不想，是不能。欠冠没清，想死都死不了。”

周无昼脸色白了一分。

裴谦却像被祝福，立刻抬手：“很好，那就按流程来。”

他把黑墙上的条款读得像在点菜：

“处置要执行。”

“执行要交付。”

“交付要签收。”

他说到最后三个字时，眼神温柔得像递糖——可递出去的是刀。

“问题来了：谁交？谁收？谁签收？”

执案官怒道：“当然是你交！”

裴谦认真地点头：“我交不了。我没有‘周’字第三笔。”

执案官噎住。

这话要是放在别的案子里，直接就是“窗口关闭”。可偏偏黑墙背后那行【失败惩罚：证物永久保全（不得处置）】像反手把刀架在所有人脖子上。

窗口一旦失败，裴谦就会被永久保全。

裴谦死不了。

而债——永远结不了。

司晦承书的雾身明显一滞。

裴谦趁热打铁，轻轻补上一句：“条款只写了‘交付’，没写‘交付什么形态’。”

周无昼心里一沉：“你又要解释？”

裴谦点头：“不解释我怎么死？”

他伸出一根手指，像在卷宗上写批注：

“交付，不一定是把你拆下来。”

“交付，也可以是——交付笔画所有权。”

司晦承书冷笑：“你想把周无昼那一笔变成你的？”

裴谦纠正：“不是变成我的。是变成‘处置证据’。”

他把“证据”两个字咬得很轻，却像把黑墙上那滴墨也咬住了。

“证据要有人签收。”

空气里一静。

那滴墨轻轻晃了一下，像在笑。

司晦承书声音阴冷：“谁签收？你吗？你是证物。”

裴谦立刻点头：“我签收也行。”

周无昼急了：“你签收等于你承认你能死，上卷会把你写成‘自毁证据’，到时你连‘合法处置’都不算！”

裴谦愣了半秒，反而眼神更坚定：“那也行。”

周无昼：“……”

执案官已经不想跟这疯子讲人话，转头朝灰雾喊：“司大人！条款写得清清楚楚，等价证据必须交付！”

司晦承书冷硬：“我只是债主。”

裴谦温柔提醒：“债主也要签收。你们自己写过：写入=签收=到场=实名闭环。”

他抬手指向周无昼头顶欠冠：“你要他那一笔当证据，你就得——到场签收。”

司晦承书阴笑：“到场可以。签收不行。签收就背执行责任。”

裴谦点头：“你不签收，就是拒绝执行。”

“拒绝执行=阻碍结案。”

“阻碍结案=窗口失败。”

“窗口失败=证物永久保全。”

他说到最后一句时，黑墙竟像被迫回忆起那行惩罚，墨迹隐隐发亮。

司晦承书沉默。

周无昼咬牙：“别逼他。他会把这笔转嫁给我。”

裴谦眨眨眼：“转嫁也得走流程。”

司晦承书果然冷笑着开口，像在宣判：

“欠冠在你头上，‘周’字第三笔本就属于你。”

“你交付，你签收，你执行处置——”

“我只负责见证。”

周无昼脸色瞬间灰败。

那是把他从欠债人直接推成刽子手。

裴谦却在旁边点头：“好主意。”

周无昼差点当场裂开：“你到底站谁那边？！”

裴谦很诚实：“我站死那边。”

他抬手，像敲响了一枚无形印章。

“但是——司晦承书，你这叫转嫁签收。”

“转嫁签收=代签。”

“代签——会被否决。”

‘叮。’

规则轻响。

《代签否决》。

司晦承书雾身猛地一僵，像喉咙里被塞进了一根看不见的钉子。

黑墙吐出两行字，冷得像刀背：

【判定：转嫁签收为代签行为】
【代签否决：强制实名签收】

下一瞬，司晦承书的雾身胸口“砰”地炸开一圈灰墨。

一道极细的“在场”印记，像烙铁一样落在他雾面上。

【署名：司晦承书】
【状态：在场（强制）】

他发出一声低哑的嘶吼，像被当庭按着签了生死状。

而周无昼头顶欠冠，利息线“咔”地断了一截，像从空中掉下细细的灰。

他怔住。

——这不是语言上的胜负。

这是规则给的“当场掉血”。

裴谦看到那截灰落下，眼神像看见了死亡证据：“好！兑现了！”

执案官脸色惨白，嘴唇都在抖：“你……你们把庭审当什么了？！”

裴谦认真纠正：“当流程。”

司晦承书咬牙切齿：“第三笔不在我手里！”

周无昼声音发哑：“也不在我手里。”

裴谦一愣：“什么意思？”

周无昼像是掏出了自己的脏底牌，艰难地吐字：“上卷扣着。那一笔是押品。它用我的‘周’字第三笔，钉着见证条款……我才会一直在场。”

裴谦听完，不但不慌，反而像见到新开的一扇死门。

“押品在上卷手里？”

“那更好。”

他抬头看那滴墨，笑得干净得过分。

“押品要取。”

“取押品=处置证据流程。”

“处置证据——必须执行者实名在场签收。”

他一句一句往黑墙里钉。

黑墙骤然一震，墨迹翻滚，像被迫补全流程：

【流程补充：取押品视为‘处置证据’】
【处置证据需执行者实名在场签收】

周无昼瞳孔骤缩：“你把上卷也往执行者位置拖？”

裴谦点头：“不拖它进来，它就会把我写成‘证：其存续’。”

他指了指胸口浮起的证纹：“你看，墨尖已经对准我了。”

那滴墨果然不再晃。

墨尖一点点下压，像锋利的针。

【21:06】

司晦承书雾身疯狂翻涌，显然在挣扎：签收背执行；不签收窗口失败；窗口失败证物永久保全；证物永久保全债永远结不了。

他恨得牙痒：“你就是个疯子。”

裴谦点头：“谢谢。”

倒计时继续。

【18:59】

黑墙上那行【等价证据：周字第三笔】忽然泛起淡淡墨光。

像卷宗深处有什么东西被拎起。

所有人都听见了那一声——

‘叮。’

像押品柜被打开的锁。

半空中，一道极细的笔锋浮现，先勾出一个“周”字的虚影。

两笔已成。

第三笔却迟迟不落。

它像一根未点名的针，缓缓转向。

先对准周无昼的欠冠。

又偏了半寸，停在司晦承书雾面前。

最后，竟然抖了一下，直指那滴墨本身。

黑墙吐出一行新的、极短的字。

【执行者候选：在场实名签收者】

裴谦笑了。

他抬起头，声音轻得像问一句家常——却像把人往刑台上推：

“周字第三笔，谁来签收？”

墨尖悬在空中。

答案只差一笔。
