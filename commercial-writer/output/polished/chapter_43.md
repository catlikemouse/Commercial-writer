# 第43章：第二个字落下（润色版）

【00:59】

墨手悬在半空。

它刚写下【司】。

那一笔像钉子，把整座问罪台的空气都钉得发闷。

周无昼头顶的欠冠边沿又崩了一角，暗墨顺着他的鬓角往下淌。他像被人拿着账本按在地上，喉咙里挤出一声低哑的笑——更像咳血。

“你满意了？”周无昼盯着裴谦，眼神像要把他从字里抠出来，“真名首字落地。接下来……你知道会发生什么。”

裴谦点头，语气还挺礼貌：“知道。下一步写第二个字。”

秦执脸色发青：“你能不能别把这句话说得像‘下一道菜上桌’？”

裴谦想了想，诚恳纠正：“更像‘下一刀落下’。”

司晦承书的雾身被那一笔反噬撕出裂口，裂口里滚出一线细墨，像血，也像欠条。

他声音发冷：“裴谦，你真以为真名写出来，你就能按流程把自己销毁？你忘了上卷【保全：异常样本需持续存续】。”

裴谦抬眼：“我没忘。”

“我只是想让它自己承认——它同意销毁。”

他抬手，轻轻敲了敲黑墙。

黑墙字仍滚烫：

【留痕未完成：需写满三字以构成可念读真名】
【提示：第二字落下 → 归位进程不可逆】

裴谦看着那两行字，像看见了死亡在排队叫号。

“你看，”他对周无昼说，“它自己写的。三字写满，可念读。”

周无昼嗓音沙哑：“可念读意味着什么？”

裴谦想都不想：“意味着大家都能念。”

他顿了顿，补上一句更像歪理但又精准得可怕的话。

“众念一响，它就不能装死。”

【00:52】

墨手缓缓抬起。

它要写第二个字。

周无昼几乎是本能地后退半步，脚跟却被问罪台的阵纹钉住，退无可退。

秦执握着腰间那块“签收印”，手指发白：“黑墙，能不能……暂停？”

黑墙回写一行冷字：

【流程不可暂停】

裴谦听得更开心了：“你看，连暂停都不让。天道都不让我休息，真贴心。”

司晦承书冷笑：“贴心？它是要把你写成永存证物，写到你连死这个字都没资格用。”

裴谦“嗯”了一声，像在认真记笔记。

“所以我才要逼它把‘同意销毁’写得更清楚。”

他说着，抬头对着那只墨手，语气甚至带点催促的温柔：

“写。”

“写快点。”

“写完三字，我们就按流程办。”

【00:45】

墨手停住了。

它像在犹豫。

——它当然犹豫。

第二字一落，归位不可逆。

归位意味着本体要更近。

更近意味着反噬更疼。

司晦承书雾身一颤，忽然开口，像在替那只手说话，又像在给它递台阶：

“黑墙写的是‘三字可念读’。可念读不等于要写完整。”

他声音里带着阴险的轻巧：“拆字也可念。比如把第二字拆成两半——”

话没说完。

裴谦抬手，指尖轻轻一点。

《代签否决》像一枚无形的印章，啪地盖在司晦承书嘴上。

黑墙字落：

【检测：代落笔/代拆字意图】
【判定：代意图成立 → 代签风险】

司晦承书胸口那枚【在场（强制）】烙印猛地一亮，像有人拿火钳夹住他的雾骨。

他闷哼一声，雾身裂口扩大。

裴谦语气平静得可怕：“你想帮它拆字。”

“在我这里，拆字也是代签。”

司晦承书咬牙：“你把所有路都堵死，你以为你堵的是我？你堵的是——”

裴谦接话：“堵的是它的逃跑路线。”

【00:41】

司晦承书忽然抬手。

他那团雾，像伸出一根看不见的指头，去掐墨手落笔的那点“墨”。

——断墨。

只要墨断了，字就写不成。

只要字写不成，真名就不成立。

只要真名不成立，归位就还能拖。

他指尖刚碰到那点墨——

“嗤——”

胸口【在场（强制）】烙印爆出一声尖响。

不是火。

是“签收”的反噬。

司晦承书整个人像被黑墙拽了一把，雾身狠狠贴向墙面，裂口里喷出一撮黑色墨屑。

那墨屑落在地上，竟化成一截残破的笔画，像断掉的骨头。

秦执瞳孔一缩：“……掉笔画了？”

裴谦眼睛一亮，像看见了战利品：“很好。会掉就证明能掉。”

司晦承书脸色第一次真正难看：“你——”

裴谦抬手，隔空点了点那截笔画碎片，语气轻飘飘的：

“想断墨？可以。”

“先实名签收‘断墨行为’。”

黑墙立刻补字，像被戳到了关键：

【补充校验：留痕须可念读且可追责】
【拆字/涂抹/遮笔/断墨 → 视为拒绝批准 → 直接执行】

“直接执行……”秦执一口气差点没喘上来，“执行什么？”

黑墙没有解释。

它只把“直接执行”四个字写得更深，像在石上刻。

裴谦却笑了。

“执行好。”

“执行就能死。”

周无昼看他像看疯子：“你就不能想想别的活法？”

裴谦认真回答：“我一直在想。”

“但我每次想活，它就把我写得更活。”

“所以我决定不想了。”

【00:31】

墨手终于落笔。

它像被黑墙的校验掐着喉咙，只能写“可念读、可追责”的那一种写法。

第二个字的第一笔落下时，问罪台上空的墨浪忽然翻涌。

不是风。

是“字”的回声。

仿佛有人在天穹里，把“司”这个姓念了一遍。

再念一遍。

越念越清。

周无昼头顶欠冠猛地一震，像被众念拉紧的绳子抽了一下，他膝盖一软，差点跪下。

秦执也被压得胸口发闷，嘴里冒出一句极不合时宜的吐槽：“……这玩意儿怎么比上宗的禁制还讲究‘群众基础’？”

裴谦点头：“群众基础越厚，杀起来越响。”

【00:24】

墨落成字。

——【晦】。

一瞬间，整个问罪台的阵纹像活了。

黑墙上那行提示字，仿佛被谁用指甲刮过，发出令人牙酸的“咔咔”声：

【归位进程：启动（不可逆）】

司晦承书的雾身猛地一缩，像被人从名字里拽住脊梁。他第一次失了镇定，声音里带了惊怒：

“不！你不能让它归位！”

裴谦抬眼，语气轻得像在问路：“已经归了。”

司晦承书厉声：“归位之后，你们所有人都会被写进它的卷宗里！你以为你是例外？你不过是它的证物！”

裴谦很认真地点头：“对，我就是证物。”

“所以我才要——销毁证物。”

这句话说出来，周无昼眼神一滞，像忽然意识到：这人是真的在用规则自杀。

裴谦后脑那只“轮盘”猛地一震。

“咔啦——”

系统红字跳出，竟不是单纯警告，而是一条冷冰冰的“补偿”：

【适应完成：真名回声抗性（入门）】
【效果：众念压迫阈值+1；归位威压对你影响减弱】
【代价：适应冷却延长（预计+18%）】

裴谦看着“阈值+1”，像捡到一块能拿去抵命的筹码。

“看，”他抬头对周无昼说，“它越不让我死，我越有办法让它按流程杀我。”

周无昼喉头发苦：“你把命当账本算？”

裴谦点头：“账本至少能对上。”

【00:17】

黑墙的字开始变。

不是改。

是“加注”。

像有人把更深的卷宗翻开，把更高层的条款往下压。

【真名回声：扩散】
【在场者：众念绑定】

裴谦看见“众念绑定”四个字，眼睛亮得像刀锋。

他抬手，直接把问题顶到最危险的解释上：

“黑墙。”

“既然众念绑定了，那批准是不是就必须在众念之下完成？”

黑墙沉默半息。

这半息里，司晦承书像是想把雾身撕开逃走，却被【在场（强制）】烙印一寸寸拉回。

周无昼想开口阻止，又被欠冠的反噬压得喉头发苦。

秦执想当和事佬，嘴一张，又闭上。

最后，黑墙落字：

【校验通过：批准闭环须在可念读众念场中完成】

裴谦轻轻“啊”了一声，像终于拿到了一纸死亡批文。

“听见没？”

他抬头看墨手，语气甚至带点礼貌的请示：

“那就请你把第三个字写完。”

【00:10】

墨手抬起。

第三字未落。

但归位的威压已经压到每个人的骨头里。

周无昼头顶欠冠开始渗出更黑的墨，像利息在加倍。

秦执喉结滚动，手掌发抖：“……第三字是什么？”

司晦承书声音发颤，却还试图嘴硬：“你们不知道第三字？那就别让它写！只要三字不成，可念读真名不成立——”

裴谦打断他，温和得像在纠正一个做错账的伙计：

“你又想代签。”

“你说不让写，就是替它拒绝批准。”

黑墙字冷冷落下：

【检测：阻断留痕 → 等同拒绝批准】

司晦承书当场噎住。

裴谦笑得更开心了。

“你看，黑墙都替我说话。”

他抬起两指，像在给死亡敬礼。

“写。”

“第三个字写出来。”

“写满三字——我们就按流程，执行。”

墨手的尖端，终于触向黑墙。

第三字的第一笔，落下。

——像一柄刀，插进所有人的命格里。

黑墙忽然弹出一行新字，像把悬念直接钉进读者眼睛里：

【第三字候选：承】
【执行条款：真名三字写满 → 当庭完成批准闭环（同意销毁）】

秦执脸色一白：“……承？”

周无昼却像被那行“当庭完成”刺中，喉咙里挤出一句近乎绝望的低骂：“它要逼我们在众念之下，亲眼看着你被——”

裴谦打断他，声音里是久违的轻快。

“被销毁。”

他抬头，看着那只墨手，像看着一把终于肯对准自己的刀。

后脑轮盘“咔啦”一声，转得比任何一次都快。

系统红字一闪而过：

【警告：署名本体归位压迫】

裴谦却只回了两个字：

“好价。”

（本章完）
