# 第三章：这里的剑怎么都在躲我？

裴谦站在广场中央，看着地上打滚哀嚎的王霸，又看了看四周那一圈圈“敬畏到快要跪下”的目光，只觉得人生灰得发亮。

他抬起两只手，十分真诚地解释：

“我……真不是故意的。”

——坏就坏在他太真诚。

在众人眼里，这句话不是解释，是轻描淡写的“我随手而已”。

“不是故意的？随手一击就把大师兄烧成炭？”
“那要是他认真……嘶！”

执法堂弟子们脸色发白，连呼吸都放轻了，像是怕惊动什么不可名状的存在。

裴谦听得更心累。

你们怕什么？我还嫌你们不够狠呢。

他脑子里只剩一个念头：

这地方不能待。

太菜了。

连个筑基期巅峰的大师兄都杀不死他，反倒被反噬烧秃了——这让裴谦产生了极其强烈的职业失望。

“不行……我得找个更厉害的地方。”

他眼神发亮，像终于看到通往“重开”之路的路标。

“哪里更危险？哪里必死无疑？”

脑海飞速翻找宗门地图，裴谦忽然想到一个传说：

宗门禁地——【万剑冢】。

传闻那里埋葬着数万把上古残剑，剑意残留，剑气不散。别说杂役，便是金丹长老心神不稳，都可能被暴走剑气绞成肉泥。

裴谦眼睛一亮。

“就是它！”

他几乎要感动得哭出来：这世上竟有如此诚实的地方。

他甚至懒得再看王霸一眼，拔腿就跑。

方向——直指后山禁地。

“快拦住他！”

赵刚嘶声大喊，声音都变了调。

但下一秒，喊声就卡在喉咙里。

拦？谁拦？

那可是把大师兄烧成炭的人。

众弟子你看看我、我看看你，最后齐齐后退半步，动作整齐得像练过。

裴谦一路狂奔，穿过山道，很快来到一片迷雾森林前。

雾像冷水一样贴在皮肤上，空气里带着金属割裂的寒意。

一块巨大的石碑立在入口，碑面刻着四个鲜红如血的大字：

【宗门禁地，擅入者死！】

裴谦盯着那个“死”字，越看越亲切。

“写得真好。”

他眼眶竟有点热。

多么醒目的承诺，多么真诚的邀请。

裴谦不再犹豫，一头扎进迷雾。

刚踏入禁地范围，周围的空气骤然变得锋利。

无数道肉眼可见的剑气在雾中穿梭，如鱼群掠过水面，发出令人牙酸的尖啸。

裴谦张开双臂，闭上眼，脸上甚至带着点陶醉。

“来吧。”

“快把我也绞碎吧。”

嗖嗖嗖——！

剑气瞬间合围，像万千刀刃同时落下。

然而就在这一刻——

那个熟悉到让他想骂人的声音，在脑海里猛地炸响。

咔哒、咔哒、咔哒！

像一枚齿轮被强行加速，硬生生要把他的脑子拧碎。

【检测到多重物理切割伤害……】
【检测到精神震荡伤害……】
【检测到剑意威压……】
【八握剑·异界神将·魔虚罗系统：全功率运转】

叮叮叮叮叮！

清脆提示音像连珠箭一样射进脑海。

【适应完成：剑气穿透】
【获得：中级剑气免疫 → 高级剑气免疫】

【适应完成：精神震荡】
【获得：剑意抗性 → 剑心通明】

【触发：环境级规则压制】
【获得：万剑臣服光环】

裴谦甚至没来得及感到痛。

剑气落在他身上，像撞上了一层看不见的壁障，全部被弹开。

更诡异的是——

它们居然在……发抖。

裴谦能清楚地感觉到，那些本该无情的剑气像受惊的野兽一样退缩。

“……”

他睁开眼，沉默良久。

“不对。”

“你们这是干什么？”

他抬起手，朝前走了一步。

雾中，土里、石缝里、树根旁，一把把残剑开始“自己动”。

锈迹斑斑的铁剑、断了半截的灵剑、插在巨石上的古剑……它们纷纷从沉眠中拔出，颤抖着浮起。

裴谦心里一喜。

终于要开始了？

下一瞬，他的笑容僵住。

这些剑不是冲他来的。

它们……在给他让路。

数万残剑分列两侧，剑尖触地，剑柄微弯，姿态整齐得像在迎接一位君王。

裴谦当场傻眼。

“砍我啊！”

他忍不住吼了一声，声音在山谷里回荡。

“这么多剑，难道就没有一把有骨气的吗？”

他不信邪，随手抓起一把插在地上的灵剑，往自己胳膊上狠狠一划。

当！

剑刃直接崩出一个缺口。

那灵剑发出一声凄厉悲鸣，像是被逼着去捅祖宗，嗖的一下飞走，插得远远的，再也不肯靠近。

裴谦：“……”

他忽然有点委屈。

我只是想死而已，你们至于吗？

就在这时，天边数道流光破雾而来。

是宗门长老。

还有掌门。

执法堂闹出这么大的动静，他们早就被惊动。更别提听说“裴谦冲进万剑冢”，众人差点当场魂飞魄散——那可是宗门根基。

可当他们赶到禁地边缘，看见眼前的景象时——

一个个惊得差点从飞剑上掉下来。

“这……这是……”

执法长老胡子都在哆嗦。

“万剑臣服？！这怎么可能？！”

“当年开山祖师也只能勉强压制剑气！”

掌门眼珠子都快瞪出来了，声音发颤。

“他……他竟能让万剑自动让路？！”

“先天剑体？不……这根本不是剑体，这是剑道至尊转世！”

有人又惊又怕地指着深处。

“看！那几把已经生出灵智的上品灵剑，竟然飞过去给他……当板凳坐！”

“天哪！那可是祖师留下的‘青冥剑’！平时我碰一下都要被震飞，现在居然在给他……挠痒痒？”

裴谦此刻正坐在一把青光流转的灵剑上，表情生无可恋。

这破剑非要往他屁股底下钻，赶都赶不走，还时不时抖两下，像在讨好。

他抬头看向禁地边缘那群目瞪口呆的老头子，心里只剩疲惫。

“我都这样了。”

“你们能不能别看了？”

“能不能……哪怕有一个人，敢冲进来杀了我？”

没有。

在“万剑臣服”的威压之下，就连元婴期的掌门都觉得心惊肉跳，脚底发虚。

裴谦叹了口气，从剑身上跳下来。

“没意思。”

“太没意思了。”

他环顾四周，语气像在点评一桌难吃的酒席。

“这破禁地，连个能打的都没有。”

无数残剑仿佛听懂了他的嘲讽，齐齐发出低哀的剑鸣，把剑尖压得更低。

而长老们听到这句凡尔赛到极致的话，几乎要窒息。

破禁地？

连个能打的都没有？

你知不知道我们平时连靠近都不敢？

裴谦抬头看了看天。

主峰方向，云层似乎比别处更厚，隐隐有雷光在深处翻滚。

他想起传闻：宗主玄清道人正在闭关冲击化神，雷劫将至。那种天威，一道劈下，魂飞魄散。

裴谦眼睛一点点亮起来。

“雷劫……”

“总不能也免疫吧？”

他越想越觉得靠谱，甚至有点热血。

下一秒，他脑海里的齿轮声突然“咔”的一顿，像被什么更大的规则按住。

裴谦：“？”

他心脏莫名跳快了一拍。

不是害怕——是期待。

终于有点像样的东西了。

他转身就走，脚步比刚才更快。

“那个谁，”裴谦指了指掌门，语气客气得像请人吃饭，“掌门是吧？带个路。”

他顿了顿，真诚补充：

“我想去见你们宗主。”

“顺便……让他劈我两下，谢谢。”
