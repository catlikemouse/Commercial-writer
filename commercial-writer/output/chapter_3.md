# 第三章：这里的剑怎么都在躲我？

裴谦站在广场中央，看着躺在地上打滚的王霸，以及周围那些敬畏的眼神，只觉得人生无比灰暗。

“我……不是故意的……”
他弱弱地解释了一句，但这只会让人觉得他更加高深莫测。

“不是故意的？随手一击就这么强？那要是故意的岂不是要翻天？”
“恐怖如斯！此子恐怖如斯！”

执法堂的弟子们已经开始瑟瑟发抖了，生怕这个“魔神”一不高兴把他们都点了天灯。

裴谦知道，这地方不能待了。
这帮人太菜了！
连个筑基期巅峰的大师兄都弄不死他，反而还被自己的火反噬。

“不行！我得找个更厉害的地方！”
“哪里更危险？哪里必死无疑？”
裴谦的大脑飞速运转。

突然，他想到了一个地方。
宗门禁地——【万剑冢】！

传说中，那里埋葬着数万把上古残剑，还有无数前辈高人留下的剑意。
就算是金丹期长老进去，如果心性不稳或者运气不好，也会被暴走的剑气绞成肉泥！

“就是它了！”
裴谦眼睛一亮，仿佛看到了通往极乐世界的金光大道。
“只要冲进去，绝对活不过三秒！”

打定主意，裴谦甚至没去管还在哀嚎的王霸，拔腿就跑。
方向——直指后山禁地！

“快拦住他！他要跑！”
赵刚见状大喊，但没人敢动。谁敢拦？那可是连大师兄都烧成炭的人！

裴谦一路狂奔，很快就来到了那片令人闻风丧胆的迷雾森林前。
这里竖着一块巨大的石碑，上面刻着鲜红的血字：
【宗门禁地，擅入者死！】

“好！这四个字写得真好！”
裴谦感动得都要哭了。多么亲切的警告，多么真诚的承诺！
“死”字多大啊！多醒目啊！

他没有任何犹豫，一头扎进了迷雾中。

刚一踏入禁地范围，周围的空气瞬间变得无比锋利。
无数道肉眼可见的剑气，如同实质般在空气中穿梭，发出令人牙酸的尖啸声。

“来吧！快把我也绞碎吧！”
裴谦张开双臂，闭上眼睛，一脸陶醉地迎接死亡。

*嗖嗖嗖嗖！*

密集的破空声响起。
无数道剑气瞬间将裴谦包围。

然而——

*咔哒……咔哒……咔哒……*

那个该死的齿轮声，又一次响了起来。
而且这一次，比以往任何时候都要急促，都要密集！

【检测到多重物理切割伤害……】
【检测到精神震荡伤害……】
【检测到剑意威压……】
【八握剑·异戒神将·魔虚罗系统全功率运转……】

*叮叮叮叮叮叮！*

一连串的清脆提示音，如同开了二倍速一样在裴谦脑海中炸响。

【剑气穿透适应完成。】
【获得：中级剑气免疫 -> 高级剑气免疫。】
【精神震荡适应完成。】
【获得：剑意抗性 -> 剑心通明。】
【获得：万剑臣服光环。】

裴谦甚至还没感觉到痛，那些剑气就已经打在他身上了。
但诡异的是，它们就像是撞上了一层看不见的屏障，或者是碰到了什么极其可怕的东西，瞬间弹开，然后……瑟瑟发抖？

是的，瑟瑟发抖。
裴谦甚至能感觉到这些剑气在害怕！

不仅如此。
随着他往里走，周围那些原本狂暴无比、见人就杀的残剑，竟然纷纷从土里拔了出来。
不是要攻击他，而是……给他让路！

一把把锈迹斑斑的铁剑，一把把灵光闪烁的断剑，甚至还有几把插在巨石上的古剑，全部都在颤抖。

它们自动分列两旁，剑尖触地，剑柄微弯。
那姿态，就像是在迎接一位至高无上的君王！

“……”
裴谦看着这一幕，彻底傻眼了。

“不是……你们这是干什么？”
“砍我啊！你们倒是砍啊！这么多剑，难道就没有一把有骨气的吗？”

他随手抓起一把插在地上的灵剑，往自己胳膊上划了一下。
*当！*
剑刃直接崩了一个口子。
那灵剑更是发出一声悲鸣，像是受了什么天大的委屈，嗖的一下飞走了，插得远远的，再也不肯靠近。

“……”

就在这时，几道流光从天边飞来。
是宗门的长老们！
还有掌门！

执法堂出了这么大的事，他们早就被惊动了。尤其是听说那个叫裴谦的“魔神”冲进了万剑冢，更是吓得魂飞魄散。
那是宗门根基所在啊！要是被毁了怎么办？

然而，当他们赶到禁地边缘，看到眼前的景象时，一个个都惊得从飞剑上掉了下来。

“这……这是……”
执法长老胡子都在哆嗦。
“万剑臣服？！这怎么可能？！”

“就算是当年的开山祖师，也只能勉强压制这里的剑气！他……他竟然能让万剑自动让路？！”
掌门更是瞪大了眼睛，一脸不可思议。
“此子究竟是什么来头？难道是传说中的……先天剑体？不，是剑道至尊转世！”

“看！那几把已经生出灵智的上品灵剑，竟然主动飞过去给他当板凳坐！”
“天哪！那可是祖师留下的‘青冥剑’啊！平时连我碰都不让碰，现在居然在给他……挠痒痒？”

裴谦坐在那把名为‘青冥’的灵剑上，一脸生无可恋。
这破剑非要往他屁股底下钻，赶都赶不走。

他看着远处那群目瞪口呆的老头子，只觉得心累无比。

“我都这样了，你们能不能别看了？”
“能不能哪怕有一个人，敢冲进来杀了我？”

但没人敢动。
在这万剑臣服的恐怖威压下，就连元婴期的掌门都觉得心惊肉跳。

裴谦叹了口气，从剑身上跳下来。
“没意思。太没意思了。”

“这破禁地，连个能打的都没有。”

他这一句话，在寂静的山谷中回荡。
无数把残剑仿佛听懂了他的嘲讽，齐齐发出一声哀鸣，把头埋得更低了。

而那些长老们，听到这句极其凡尔赛的话，一个个都快要窒息了。
“破禁地？连个能打的都没有？”
“这也太狂了吧！”
“不过……他好像真的有狂的资本啊……”

裴谦抬头看了看天。
听说宗主最近在闭关冲击化神期？
据说那种雷劫特别猛，稍微碰一下就会魂飞魄散？

“要不……去找宗主玩玩？”
裴谦眼睛一亮。
“雷劫总不能免疫了吧？那可是天威啊！”

想到这里，他也不管身后那群还在石化的长老，拍了拍屁股上的灰，大步朝着宗主闭关的主峰走去。

“那个谁，掌门是吧？带个路，我要去见你们宗主！”
“我想让他劈我两下，谢谢！”
