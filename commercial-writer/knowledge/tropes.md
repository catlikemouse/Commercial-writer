# 常见网文流派与套路解析 (Commercial Tropes Database)

## 1. 重生流 (Rebirth)
**核心爽点：** 信息不对称、弥补遗憾、先知先觉。
- **开局**：前世悲惨/遗憾 -> 重生回关键节点。
- **金手指**：前世记忆（功法、股市、彩票、未来大事件）。
- **节奏**：快速切入，利用前世经验迅速积累第一桶金/第一波实力。
- **忌讳**：重生后性格依然软弱，或者纠结于前世情感（除非是虐文）。

## 2. 系统流 (System)
**核心爽点：** 数据化反馈、付出即回报、强制性任务带来的冲突。
- **开局**：获得系统（签到/抽奖/任务/加点）。
- **金手指**：系统本身（提供资源/技能/属性点）。
- **节奏**：任务驱动，系统发布任务 -> 主角完成 -> 获得奖励 -> 实力提升。
- **忌讳**：系统太智能（喧宾夺主）、任务惩罚太重（抹杀）、系统毫无存在感。

## 3. 无敌流 (Invincible)
**核心爽点：** 绝对碾压、扮猪吃虎、高人风范。
- **开局**：满级大佬新手村游历/隐居。
- **金手指**：自身无敌的实力/背景。
- **节奏**：反派挑衅 -> 主角无视 -> 反派作死 -> 主角一指头碾死 -> 众人震惊。
- **忌讳**：主角被小角色长期压制（憋屈）、强行降智（为了装逼而装逼）、缺乏期待感（因为已经无敌了）。
- **解决**：通过收徒、建设势力、解开更高级别的秘密来维持剧情张力。

## 4. 凡人流 (Mortal Cultivation)
**核心爽点：** 谨慎稳健、步步为营、杀人夺宝、以弱胜强。
- **开局**：资质平庸，获得神秘小瓶/残缺功法。
- **金手指**：催熟灵药/辅助修炼/神秘空间。
- **节奏**：慢热，注重细节描写，资源获取困难 -> 冒险 -> 提升 -> 换地图。
- **忌讳**：主角圣母心泛滥、轻易相信他人、没有危机感。

## 5. 迪化流 (Misunderstanding)
**核心爽点：** 脑补怪、跨服聊天、误会带来的笑料与成就。
- **开局**：主角由于某种原因（实力强/运气好/系统坑）被误认为是绝世高人。
- **金手指**：运气/被动技能/特效。
- **节奏**：主角做普通事 -> 配角过度解读 -> 造成惊人后果 -> 配角更加崇拜 -> 主角一脸懵逼。
- **忌讳**：逻辑完全无法自洽、笑点尴尬、主角真傻（应该是大智若愚或信息差）。

## 6. 签到流 (Sign-in)
**核心爽点：** 零成本高回报、苟道发育。
- **开局**：被发配/隐居某地，获得签到系统。
- **金手指**：每日/特定地点签到获得神级奖励。
- **节奏**：日常签到 -> 实力暗中暴涨 -> 外界危机来临 -> 主角出手秒杀 -> 继续苟。
- **忌讳**：过早暴露实力、签到奖励同质化、缺乏主线剧情（变成流水账）。

## 7. 退婚流 (Broken Engagement)
**核心爽点：** 莫欺少年穷、打脸前任、三十年河东三十年河西。
- **开局**：天才陨落/废柴 -> 未婚妻上门退婚 -> 定下三年之约。
- **金手指**：戒指老爷爷/神秘血脉觉醒。
- **节奏**：压抑（退婚羞辱） -> 爆发（休妻/约定） -> 苦修 -> 履行约定（打脸）。
- **忌讳**：前任洗白太快、主角一直沉浸在仇恨中无法自拔。
