# 月入5000+ 商业策略 (Income Strategy)

## 1. 收入公式 (The Formula)

> **收入 = 基础全勤 + 订阅分成 + 免费广告分成 + 礼物打赏**
>
> 目标：**5000元/月**
>
> - **全勤保底**: 1000-1500元 (前提: 日更4000+，不断更)。
> - **流量分成**: 2000-3000元 (需有效完读率 > 30%，即每100个读者，有30个能看到最新章节)。
> - **礼物/订阅**: 500-1000元 (粉丝粘性)。

## 2. 流量密码：留存率 (Retention Rate)

**第一关：黄金三章 (前3章)**
- **目标**: 留存率 > 50%。
- **核心**: 期待感建立。必须有明确的主角目标和金手指爽点。
- **战术**: 必须有 **"Hook"**（钩子）。每一章结尾都要让读者急不可耐地点下一章。

**第二关：首秀/推荐期 (10万字左右)**
- **目标**: 留存率 > 30%。
- **核心**: 第一个大高潮的完成度。此时通常是一个小卷终，必须要有酣畅淋漓的打脸/升级/收获。
- **战术**: 在8-9万字时开始铺垫大高潮，确保在10万字左右爆发，拉高完读率数据。

**第三关：上架/付费点 (20-30万字)**
- **目标**: 付费率/完读率不暴跌。
- **核心**: 长线悬念的吸引力。此时单纯的爽点已经不够了，必须靠剧情悬念（身世、终极BOSS、世界真相）吊住读者。
- **战术**: 在上架前3章，抛出一个惊天大悬念或处于绝境之中，让读者非看不可。

## 3. 防崩坏：稳定更新 (Consistency)

- **日更**: 4000字是及格线，6000字是推荐线。断更是大忌。
- **存稿**: 永远保持3-5章存稿，应对突发情况。
- **水文**: 偶尔水一章过渡可以，但绝不能连续水。
  - *技巧*: 如果卡文，写主角整理收获、与配角互动、升级系统面板，这些也是读者爱看的"爽点"，不算纯水。

## 4. 平台特异性 (Platform Specifics)

**番茄 (免费阅读)**:
- **核心**: 完读率（广告展现）。
- **策略**: 节奏极快，三章一个小高潮，十章一个大高潮。钩子必须密集。
- **忌讳**: 虐主、文青病、长篇大论的心理描写。

**起点 (付费阅读)**:
- **核心**: 订阅（付费意愿）。
- **策略**: 铺垫扎实，人物立体，世界观宏大。读者愿意为好故事买单。
- **忌讳**: 逻辑硬伤、无脑小白文。
