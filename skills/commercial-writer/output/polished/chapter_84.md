# 第84章：三分钟，抽你个实名（润色版）

倒计时像一根冷针，扎进每个人的喉咙。

【剥离验收倒计时：03:00】
【执行者抽取条件：一条“实名”】【

黑纸门的字很轻。

轻到像一句“自愿”。

可所有人都知道——这东西从不讲自愿，它只讲追责。

终署名脸色白得发青；按停人握着银钉，指节一寸寸勒紧；监管者影子站在最暗处，指尖那点暗金墨痕像活的虫，随时要爬出一个“永”。

裴谦把那片污染笔画碎片攥在掌心。

黏、冷、脏。

像前世最后一页合同的油墨：签了就认，认了就死。

“抽实名？”裴谦抬眼，语气平得吓人，“谁想当执行者？”

终署名几乎是本能后退：“别乱来！实名不是墨，是命格锚点。被抽走——就被写成执行者，背因果、背追责、背一切！”

裴谦点头：“那就对了。”

按停人暴怒：“对个屁！你想把我们也拖下去？！”

裴谦看着他，像看一枚钉子：“我下去的时候，你们问过我？”

黑纸门像听见“问责”，纸皮微微一颤，补了一行：

【见证者优先：可追责者】

终署名喉头一紧：“它又在挑最方便背锅的。”

裴谦笑意薄如刀：“那就让它挑错。”

他伸手，指尖点在“实名”二字上。

《零笔》一转，0.2息的空感像冰针扎进字缝。

他没把“实名”空掉——那样规则会像狗一样立刻叼回“裴谦”。

他空掉的是“默认答案”。

黑纸门一抽，纸面起褶，像被当众扒了裤子。

下一行字被逼出来：

【实名=可念读且可追责的实名】

按停人骂声卡住：“可念读？”

裴谦淡淡道：“念不出来，追不了责。追不了责，制度不爱。”

他抬下巴，点向那团影子。

“比如它。”

监管者影子指尖暗金墨痕一震，声音低冷：“你敢逼我落名？”

“我敢。”裴谦举起污染碎片，“你们看清楚：它代签过，掉了脏墨。脏墨可验收、可剥离。剥离要见证者签收——门已经回填我了。”

他顿了顿，像把刀往里推。

“我签。”

“但我签之前——先把验收责任人写出来。”

黑纸门翻页声响起，像被逼着吐条款。

【新增：剥离需出具《验收单》】
【验收单需填写：验收责任人】

裴谦把指尖落在“验收责任人”一栏。

《零笔》再次发动。

他空掉的不是字。

是那句最恶心的护身符：**集体**。

——正笔院最爱用“集体”逃责。追责抓雾，雾就永远不会疼。

“嗡——”

字段空了一瞬。

规则厌恶空，立刻回填：

【验收责任人：终署名（实名）】

终署名瞳孔骤缩，像被当众钉在墙上。

他想撤笔。

门边铁规亮得刺眼：

【撤笔=赖账=追责翻倍】

终署名喉头滚动，竟不敢动。

按停人抬银钉要按停这一切。

银钉还没落下，黑纸门又补一条：

【按停动作需实名签收】

按停人动作硬生生僵住，像手腕被无形绳勒死。

他咬牙切齿：“你们这些写字的……真他妈一脉相承。”

裴谦看都不看他：“别骂。制度记仇比你快。”

倒计时跳到【02:12】。

终署名成了验收责任人，就意味着——

验收失败，他背。

验收成功，剥离后的后果，他也得签。

裴谦把污染碎片往门上一贴。

“验收对象：污染笔画碎片。”

黑纸门像被塞进证据，立刻回写：

【验收对象：污染笔画碎片】
【验收标准：可追责、可剥离、可终止】

终署名冷笑：“‘可终止’……它又想偷换成‘永终止你’。”

裴谦眯眼：“它敢写，我就敢让它疼。”

他转向那行要命的字。

【执行者抽取条件：一条“实名”】【

裴谦把“条件”两个字，往最危险的解释上顶：

“条件=资格校验。”

“要抽执行者，就得可追责。可追责，就得——当庭到场，亲笔签收。”

黑纸门停了一息。

像制度在咀嚼“追责”这口肉。

随后不情不愿补上：

【新增：执行者候选需当庭到场亲笔签收】

终署名脸色更白。

监管者影子也沉默。

因为这句话等于把刀从裴谦脖子上挪开了一寸——

挪到了他们身上。

倒计时跳到【01:29】。

按停人咬牙：“你到底想让谁签？”

裴谦指向影子：“你。”

影子冷笑：“想得美。”

它指尖暗金墨痕一颤，竟先试探着往验收单上落——

落的不是字。

是一团遮蔽的墨点。

墨点要把“实名”字段涂成不可念读。

终署名眼皮狂跳：“它要躲校验！”

裴谦没吼。

他只轻轻一句：“代写=代签。”

黑纸门像被戳到痛点，瞬间回写：

【校验：遮蔽/不可念读=拒绝追责=不合规】
【触发：代签否决】

“嗡——”

反噬像铁针扎进那团墨点。

墨点当场烂掉，裂成一条脏线，哗啦掉在地上。

影子的指尖也抖了一下。

它第一次像疼。

裴谦捡起那条脏线，嘴角挑起：“想躲？躲不了。”

倒计时只剩【00:58】。

裴谦把“抽取条件”四个字再拧一刀。

“抽取——不是抽到谁的整名。”

“是抽取一条实名。”

“既然是一条，就可以是——真名首字。”

终署名倒吸凉气。

影子指尖的暗金墨痕剧烈震动。

裴谦盯着它，声音轻得像问候，却比刀更狠：

“首字，够追责。”

黑纸门“刷”地回写校验：

【实名最小单位：可念读首字】

影子终于动了。

它抬手，像被逼到墙角的兽。

暗金墨痕落下一个字。

——“司”。

首字落下的一瞬，黑纸门像闻到血，立刻锁死：

【实名：司】
【执行者候选：司（到场）】
【候选签收：已成立】

终署名失声：“它把自己写进候选？！”

裴谦眼神却更冷。

他知道：它不是认输。

它是换一种活法——

把自己写进流程，才能继续咬人。

裴谦把污染碎片死死按住，像按住蛇的七寸。

“验收开始。”

黑纸门翻页。

【验收：通过】
【剥离：启动】

“嘶——”

污染碎片像被撕开一层皮。

一截黑纹从裴谦脚踝上被硬拽出来，啪地落在纸面，化成细细的脏线。

裴谦胸口一松。

可下一秒，他看见更深处的条款冒头，像另一个绞索：

【剥离成功→样本可调用等级：提升】

终署名脸色惨白：“它要把你标成更好用的工具。”

裴谦笑了一下，笑里没有温度。

“那就让它用得更痛。”

倒计时归零。

黑纸门最后补了一行，字小得像刀尖藏在纸缝：

【下一次调用：不再预告】

还没完。

门又补了一行更小的：

【已记录：可抽取首字】
【抽取目标：在场最可追责实名（优先）】

按停人脸色瞬间变了。

终署名背脊发寒。

影子的指尖暗金墨痕却在发光——

它没死。

它只是露了一个首字。

它还会来。

裴谦低头看脚踝。

那条黑纹少了一截，却留下更清晰的记录痕。

像有人用针，在他皮肤里刻了一个字。

——“司”。

裴谦抬眼，声音很轻：

“司。”

“我记住你了。”

黑纸门翻了一页。

这一页是空白。

空白得让人发怵。

仿佛下一次，它连三分钟都懒得给——

直接抽走你嘴里能念出的那一个字。
