# 第75章：验收庭上，先斩担保人（润色版）

终笔堂的“明日”，从来不是日升月落。

是翻页。

墨灯一排排亮起，像黑暗里睁开的一串眼睛。

裴谦被银纹拖进堂时，案卷已经先写好了结局——

【担保验收庭】
【验收项：担保人须当庭签收《处决窗口条款》】
【拒签/拖延/遮蔽：视为违约，先斩担保人以止损】

“先斩”两个字，暖得像火。

按停人走在旁听位，银钉敲地，声音冷得发脆：“你笑什么？先斩他，你也未必能死。”

裴谦点头：“我不确定。但至少——不是我先被写成永存。”

按停人嗤笑：“你连解释都解释不了，只剩一张嘴。”

“嘴也够。”裴谦说。

他还剩0.2息。

够把你们的流程咬出血。

终署名已经在台前。

他袖口干净得像没沾过任何人的命，可袖底的终责印却烫得发亮——那是“可追责优先”在催他上前。

裴谦盯着他，心里只有一句：

你也会怕。

案台中央浮出一卷新的条款。

纸面只写四个字：

【处决窗口】

下方空白却像四个刀鞘：

【处决对象：__】
【执行者：__】
【见证者：__】
【终止条件：__】

终署名淡淡道：“验收很简单。我签收，担保成立，续命继续。”

按停人接口：“按流程，签收即可，不必写满字段。”

裴谦笑了：“签收即可？”

他抬手点了点那四个字。

“你们写的是‘处决’。”

“不是‘续命确认’。”

终署名眼神一沉：“处决窗口，是续命的合规出口，防你钻空子。”

“出口。”裴谦点头，笑意更轻，“那当然要开。”

堂内的影子里，监管者的声音冷漠落下：

“验收开始。”

“担保人——签收。”

终署名抬笔。

笔尖落下的一瞬，案台边缘忽然撕出一条暗栏，像纸底抽出的骨：

【签收校验：签收人须可追责（实名/在场/亲笔）】

终署名的笔顿了一下。

他不写真名。

他要写上卷的“墨符”——不可念读、不可追责。

遮蔽一成，刀就不会落到他身上。

按停人嘴角微扬，银钉轻轻一抬，像准备把这条暗栏按死。

裴谦却比他快。

0.2息。

《零笔》。

“空。”

他空掉的不是终署名。

他空掉的是暗栏里最要命的一个字——

【可追责：__】

嗡。

案台颤了一下。

下一息，熟悉的提示浮起：

【可追责优先：自动补全】

暗金墨滴从天而降，不偏不倚砸在终署名那道遮蔽笔意上，像一滴血砸进伤口。

【可追责：终署名（在场实名亲笔）】

终署名袖底的终责印轰然发烫。

他手腕一抖，笔尖在纸上拖出一道细痕——不是字，是刀口。

按停人脸色瞬间沉下去，银钉落下想按。

可银钉刚触到案台，就“咔”地一声回弹。

一行新的补条款直接回填，像当众拔了他的牙：

【旁听不得插手验收】

银钉尾端裂出一丝细纹。

按停人第一次失了节奏。

裴谦看了他一眼，语气平静得像在念账：“签得多，就跑不了。今天你只是旁听。”

秦零站得更远，脸白得像纸：“那……处决窗口，真的要开？”

裴谦没有嘲笑他。

他只是低声说：“我怕你们把我写成工具，写到我自己都不知道我是谁。”

终署名压低声音：“你想把处决对象写成谁？”

裴谦抬头，笑得干净：“写我。”

按停人冷笑：“你以为你能写？你是编号，空名存续。自动补全会优先填最可追责者。”

裴谦点头：“所以我更需要你。”

他盯着终署名，一字一顿把话塞进对方喉咙里：

“你担保我不死。”

“那你就先签收一个——能让我死的窗口。”

终署名眼神发冷：“你想让我背处决因果？”

“不是想。”裴谦轻声说，“是你自己写的。”

他慢慢念出昨天那句：

“拒签/拖延/遮蔽——视为违约。”

“先斩担保人。”

堂内静得可怕。

监管者补了一句，冷得像判词：

“拖延，计入拒签。”

终署名的笔停在纸面上。

他如果签——处决窗口一开，刀先落他。

他如果不签——违约，刀也先落他。

裴谦看着那支笔，心里忽然生出一种荒谬的快意。

你们写了这么多年别人的命。

今天，轮到你们自己。

终署名缓缓抬眼。

那一眼第一次像“活人”。

“你要我怎么签？”他声音低沉。

裴谦笑了。

“很简单。”

“你当众念你的真名。”

“用真名签收。”

“让众念场听见——让‘可追责’变成你自己背的那根绳。”

他停顿了一下，补上最后一刀：

“你不念——就违约。”

“先斩担保人。”

案台上，那四行空白字段像四张嘴，等着吞下第一个名字。

终署名的嘴唇动了一下。

堂外墨灯齐齐一暗。

像在等他开口。
