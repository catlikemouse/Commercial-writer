# 第六章：这毒药怎么有点甜？

青云宗，丹峰。

这里是整个宗门最神秘、也是最危险的地方之一。常年笼罩在五颜六色的烟雾之中，时不时还能听到不知名的爆炸声和凄厉的惨叫声（其实是炸炉后的哀嚎）。

而今天，丹峰的气氛格外凝重。

“快！封锁后山禁地！”
“所有弟子退后三里！戴上避毒面具！”
“开启护山大阵！千万不能让毒气泄露出去！”

丹峰峰主，人称“鬼手丹魔”的刘长老，此刻正一脸绝望地跪在自己的炼丹房外。

就在刚才，他试图炼制传说中的上古丹药“九转还魂丹”，结果因为配方比例失调（手抖多加了一味断肠草），导致炼丹炉发生了史无前例的大爆炸。

但这还不是最要命的。
最要命的是，这一炉失败的药渣，并没有消散，而是化作了一滩漆黑如墨、散发着诡异甜香的粘稠液体。

这玩意儿剧毒无比！
仅仅是逸散出来的一丝香气，就让方圆十里的花草树木瞬间枯萎。就连路过的一只筑基期灵鹤，闻了一口之后，也直接两眼一翻，栽倒在地，口吐白沫。

“完了……全完了……”
刘长老面如死灰，“这是传说中的‘万毒噬心液’啊！一旦扩散开来，整个青云宗都要遭殃！我就是宗门的千古罪人啊！”

周围的丹峰弟子们也是一个个吓得面无人色，瑟瑟发抖。
这种级别的毒，别说是他们，就算是元婴期的太上长老来了，也不敢轻易触碰啊！

就在所有人都束手无策、准备等死的时候。

一道充满活力的身影，突然出现在了视野中。

“让让！都让让！前面发生什么好事了？”

裴谦一路小跑着过来，脸上洋溢着期待的笑容。
自从上次雷劫没死成之后，他郁闷了好几天。太上长老的待遇虽好，但他不是来享福的，他是来重开的啊！

听说丹峰出了大事，好像是有什么恐怖的东西泄露了，他立刻马不停蹄地赶了过来。

“站住！前面是禁地！”
一名弟子想要阻拦，但看到裴谦身上挂着的那个闪瞎眼的“太上长老”玉牌，吓得赶紧缩回了手，“拜……拜见太上长老！”

裴谦根本没理他，直接冲进了封锁圈。

一股浓郁的甜香扑鼻而来。

“好香啊！”
裴谦深吸一口气，只觉得神清气爽（并没有）。

刘长老看到有人闯进来，原本还想喝止，但看清来人之后，顿时愣住了。

“太上长老？您怎么来了？”刘长老像是抓住了救命稻草，“快！太上长老神功盖世，一定有办法封印这团毒液！”

裴谦看着那团还在咕嘟咕嘟冒泡的黑色粘液，眼睛瞬间亮得吓人。

“这就是传说中的剧毒？”
裴谦指着那团东西，兴奋地问，“真的能毒死人吗？那种七窍流血、瞬间暴毙的那种？”

刘长老连连点头，眼泪都要下来了：“是的！太上长老！这毒液恐怖至极，只要沾上一滴，就会化为脓血！哪怕是闻上一口，也会经脉尽断而亡啊！”

“太好了！”
裴谦一拍大腿，“这不就是我梦寐以求的东西吗！”

终于找到了！
比雷劫更靠谱的死法！
毒死！

这种死法多优雅，多干脆！而且听起来威力很大，肯定能破防！

“那我就不客气了！”
裴谦二话不说，直接扑到了那团黑色粘液面前。

在所有人惊恐的目光注视下。
他伸出手，捧起了一大把还在冒烟的毒液。

“太上长老！不可啊！”刘长老撕心裂肺地喊道，“这不能用手碰啊！手会化掉的！”

然而，裴谦不仅碰了，他还……张开了嘴。

咕嘟！

一口闷。

“……”

整个世界安静了。
连风都停止了吹动。

刘长老张大了嘴巴，下巴差点掉到地上。
周围的弟子们更是眼珠子瞪得像铜铃，一个个像是被施了定身法。

他们看到了什么？
太上长老……竟然把那团足以毒死全宗门的“万毒噬心液”……给喝了？
喝了？！
而且还是像喝八宝粥一样，喝得津津有味？

“味道怎么样？”
一个不知死活的弟子下意识地问了一句。

裴谦砸吧砸吧嘴，细细品味了一下。
“嗯……口感有点像芝麻糊，稍微有点苦，还有点烫嘴。”
他又捧起一把，继续往嘴里塞，“不过回甘还不错，挺甜的。”

刘长老：“……”

此时此刻，裴谦的脑海里，那个熟悉的齿轮声再次响起。

*咔哒……咔哒……*

【警告！警告！检测到致死级混合毒素侵入体内！】
【毒素成分分析：断肠草、鹤顶红、七步倒、腐骨灵水……】
【系统判定：虽然很难吃，但确实能死人。】

裴谦心中狂喜。
来了！终于来了！
这次总该死了吧？这可是混合剧毒啊！

然而——

【八握剑·异界神将·魔虚罗系统启动……】
【正在解析毒素规则……】
【解析完成。】

【获得：万毒不侵体（初级 -> 中级 -> 高级 -> 究极）。】
【获得被动：毒素转化。所有摄入体内的毒素，将自动转化为精纯灵力。】
【获得成就：绝命毒师。】
【代价提示：适应冷却延长（+12%）。】
【代价提示：天道标记活跃度上升。】

*叮！*

随着一声清脆的提示音，裴谦感觉体内那一股原本还在肆虐的灼热感，瞬间变得清凉无比。
紧接着，这股清凉的气流顺着经脉流转全身，最后汇聚丹田。

轰！
他的修为，竟然直接突破了！
虽然他并没有修炼过任何功法，但在系统的强行灌注下，他的丹田第一次成形，灵力直接定阶——筑基期！

“嗝——”
裴谦打了个饱嗝。
这一次，吐出来的不是黑烟，而是一股五颜六色的毒气。

毒气喷在旁边的一块巨石上，那块坚硬的花岗岩瞬间被腐蚀成了一滩黄水。

“……”
众人看着那滩黄水，再看看毫发无伤、甚至还在打饱嗝的裴谦，只觉得头皮发麻。

连石头都化了！
他的胃是铁做的吗？
不，就算是铁胃也该化了啊！

裴谦摸了摸肚子，一脸失望。
“又没死成？”
“不仅没死，好像还……吃撑了？”

他看着剩下的大半团毒液，顿时觉得索然无味。
“这玩意儿不行啊，除了有点甜，一点杀伤力都没有。”

裴谦嫌弃地擦了擦嘴，站起身来，看着已经石化了的刘长老。
“那个……老刘啊，你这手艺不行啊。”
裴谦语重心长地批评道，“这毒药做得太失败了，不仅毒不死人，还容易让人发胖（灵力涨得太快了）。下次记得多加点砒霜，少放点糖。”

说完，裴谦摇摇头，背着手走了。
留下一群人在风中凌乱。

“……”

良久之后。

刘长老突然扑通一声跪在地上，对着裴谦离去的背影疯狂磕头。

“神迹！这是神迹啊！”
刘长老激动得满脸通红，浑身颤抖，“太上长老这是在以身试毒！他在用这种方式告诉我，所谓的‘剧毒’，只要境界到了，皆可化为补品！”

“我悟了！我终于悟了！”
刘长老仰天大笑，状若疯癫，“原来我的炼丹术一直无法突破，就是因为我不够大胆！我不够疯狂！只有像太上长老那样，视剧毒如甘霖，才能炼出真正的绝世神丹！”

“太上长老，不仅是修为通天，这丹道造诣，更是早已达到了‘万物皆丹’的无上境界啊！”

一旁的弟子们看着陷入癫狂的峰主，又看看裴谦远去的背影，眼中的敬畏之色更浓了。

“原来太上长老是丹道宗师！”
“恐怖如斯！把万毒噬心液当糖水喝，这得多高的境界？”
“以后谁再说太上长老是靠运气上位的，我第一个跟谁急！”

……

另一边。

裴谦一边走，一边感受着体内那股因为“吃毒”而暴涨的灵力，想死的心都有了。

“造孽啊……”
“我想死，不想变强啊！”
“这破系统，怎么连喝毒药都能变成升级？还有没有天理了？还有没有王法了？”

就在这时，他路过了宗门的【兽峰】。
远远地，他就听到一阵震耳欲聋的虎啸声。

吼——！

那声音充满了暴虐和杀气，听得普通弟子两股战战。

裴谦的脚步停下了。
他的眼睛再次亮了起来。

“等等……”
“毒不死我，被老虎咬死总行了吧？”
“物理撕咬伤害！这可是实打实的！”

裴谦看着兽峰的方向，嘴角露出了诡异的笑容。
“小猫咪，我来了！”
“一定要牙口好一点啊，别让我失望！”

（本章完）
